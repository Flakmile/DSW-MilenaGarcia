Etiquetas básicas de HTML, atributos y valores

Si deseas conocer mas de las etiquetas de HTML aca te daremos algunos tips de ellas.

Las etiquetas de HTML son bloques de código que dan una instrucción al navegador de como deben interpetar el contenido entre dichas etiquetas, son indicadores de lenguaje de marcas de hipertexto las cuales le dan formato y estructura al contenido de las paginas web.

Es de recalcar que HTML no se considera un lenguaje de programación dado que no crea funciones dinamicas, pero determinan cómo se verá un sitio web.

Las etiquetas HTML comienzan siempre con el símbolo «<» y finalizan con el símbolo «>»
