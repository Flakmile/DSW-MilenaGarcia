Etiquetas básicas de HTML, atributos y valores

Bienvenidos... Si deseas conocer mas de las etiquetas de HTML aca te daremos algunos tips de ellas.

Las etiquetas de HTML son bloques de código que dan una instrucción al navegador de como deben interpetar el contenido entre dichas etiquetas. Las etiquetas HTML comienzan siempre con el símbolo «<» y finalizan con el símbolo «>»
